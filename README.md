# 芝士什么
这是小沙雕做的用来混淆java字节码的混淆器

# 支持啥
小沙雕支持：
- 字符串混淆
- 临时变量删除/重命名
- 数字混淆
- 动态调用混淆
- 垃圾代码混淆
- 垃圾try catch block混淆
- 垃圾switch case block混淆

- [!] 在类文件版本52 (JDK1.8)上食用效果更佳.jpg
